# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh


para correr el proyecto se usa;
npm run dev

despues de subir hacer cosas al proyecto, es recomendable ejecutar (cada cierto tiempo)
npm run build
sirve para que el proyecto se hostee mejor

el link de la pagina se encuentra en el github, el cual es el link generado por vercel
el cual es la pagina que brinda el hosting de la pagina
todo esta a mi correro personal, pero si quieren que se pueda a otro correo, les toca borrar el vercel (los archivos) y hacer todo el protocolo, lo mismo con firebase

Componentes: un componente es un pedazo de pagina aislado en un archivo, de manera que cuando se quiera pegar en alguna parte simplemente aparezca, mi manera de pensarlo es como si fuera un objeto, que cuando se quiere implementar en algun sitio, solo la invoco y ya.
